# AI Prompt Log :-

## Project Name

Dev Detective - GitHub Profile Finder

## Objective

Build a responsive GitHub Profile Finder using HTML, CSS, JavaScript, and the GitHub REST API.

---

## Prompt 1

How to create a responsive GitHub profile search UI using HTML and CSS?

### Outcome

Created the responsive layout with a search bar, profile card, repository section, and battle mode.

---

## Prompt 2

How to fetch GitHub user data using the Fetch API and Async/Await?

### Outcome

Implemented asynchronous API calls to retrieve GitHub user profile information.

---

## Prompt 3

How to display GitHub API JSON data dynamically in the DOM?

### Outcome

Displayed avatar, name, username, bio, join date, company, location, portfolio, followers, following, and public repositories.

---

## Prompt 4

How to handle loading and error states while fetching API data?

### Outcome

Added a loading spinner while waiting for API responses and displayed a "User Not Found" message for invalid usernames.

---

## Prompt 5

How to fetch and display the latest five GitHub repositories?

### Outcome

Fetched repository data using the `repos_url` endpoint and rendered the latest five repositories with clickable links.

---

## Prompt 6

How to format GitHub ISO date into a readable format using JavaScript?

### Outcome

Converted ISO date strings into a human-readable format (e.g., 25 Jan 2023).

---

## Prompt 7

How to compare two GitHub users using Promise.all()?

### Outcome

Implemented Battle Mode to fetch two GitHub profiles simultaneously using `Promise.all()`.

---

## Prompt 8

How to calculate the total stars of a GitHub user?

### Outcome

Used the `reduce()` method to calculate the total `stargazers_count` across all repositories.

---

## Prompt 9

How to determine the winner between two GitHub users?

### Outcome

Compared total stars and displayed Winner and Loser cards dynamically.

---

## Prompt 10

How to improve code readability and organize JavaScript into reusable functions?

### Outcome

Refactored the project into modular functions for fetching data, rendering UI, handling loading states, and managing errors.
