# Dev Detective 🔍

A modern GitHub Profile Finder built using **HTML, CSS, and Vanilla JavaScript**. 
   
   This project uses the **GitHub REST API** to search GitHub users, display their profile information, fetch repositories, and compare two developers in **Battle Mode**.

---

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.

Or deploy using Vercel or Netlify.

---

## ⚔ Battle Mode :-

Battle Mode compares two GitHub users.

It calculates:

* Total Repository Stars
* Public Repositories
* Followers

The user with the higher total stars is declared the Winner.
